# 核对清单{#checklists}

以下清单重点说明了一些主要问题 — 请阅读与测试相关的其他部分以查看完整的详细信息。

| 检查 | 响应 | 操作 | 所有者 | 到期日期 |
|---|---|---|---|---|
| 是否已建立明确的定期报告机制？ |  |  |  |  |
| 状态会议是否涉及关键参与者（来自测试团队）？ |  |  |  |  |
| 是否明确界定了测试职责（尤其是对客户而言）？ |  |  |  |  |
| 客户是否有履行测试承诺所需的经验？ |  |  |  |  |
| 是否已选择所有工具（测试和跟踪）？ |  |  |  |  |
| 是否已安装上述工具并根据需要培训用户（主要是项目团队）？ |  |  |  |  |
| 是否已制定明确、全面的测试计划？ |  |  |  |  |
| 测试计划是否已经过相关各方的审查？ |  |  |  |  |
| 验收测试是否明确定义？ |  |  |  |  |
| 验收试验是否得到各方的正式接受？ |  |  |  |  |
