# 删除域 {#delete-a-domain}

使用“域管理”页可将现有域标记为过时的。

如果创建的域与已删除的域同名，则删除的域及其包含的信息将恢复。 无法删除DefaultDom域。

1. 在管理控制台中，单击 **[!UICONTROL 设置>用户管理>域管理]**。
1. 选中要删除的域的复选框，单击“删 **[!UICONTROL 除]**”，然后单击“确 **[!UICONTROL 定”]**。
