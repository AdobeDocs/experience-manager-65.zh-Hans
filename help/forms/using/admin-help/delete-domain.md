# 删除域 {#delete-a-domain}

使用“域管理”页可以将现有域标记为已过时。

如果创建的域与已删除的域同名，则已删除的域将与其包含的信息一起恢复。 无法删除DefaultDom域。

1. 在管理控制台中，单击 **[!UICONTROL “设置”>“用户管理”>“域管理”]**.
1. 选中要删除的域的复选框，然后单击 **[!UICONTROL 删除]**，然后单击 **[!UICONTROL 确定]**.
