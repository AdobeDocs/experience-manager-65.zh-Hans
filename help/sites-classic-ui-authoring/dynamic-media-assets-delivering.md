# 交付Dynamic Media资产{#delivering-dynamic-media-assets}

如何交付Dynamic Media资产（包括视频和图像）取决于网站的实施方式。

使用Dynamic Media，您有以下几个选项：

* 如果您的网站托管在Adobe Experience Manager上，则您希望将Dynamic Media资产直接添加到您的页面。
* 如果您的网站未Experience Manager，请参阅触屏优化用户界面中的[交付Dynamic Media资产](/help/assets/delivering-dynamic-media-assets.md) 。

有关更多信息，请参阅下列主题：

* [将Dynamic Media资产添加到页面](/help/sites-classic-ui-authoring/dynamic-media-assets-adding-to-page.md)
