# 问题与答案论坛功能{#q-a-forum-feature}

## 简介 {#introduction}

问题与答案(QnA)论坛功能为社区成员提供了一个提问和回答问题的区域。 它允许成员：

* 创建新问题
* 添加内联图像（支持拖放）
* 视图和回答问题
* 搜索问题
* 帮助审核问题与答案内容
* 确定最佳答案
* 将问题解答问题从一个页面移到另一个页面

文档描述：

* 向AEM站点添加问题与解答论坛功能。
* 组件的配置 `QnA`设置。

## 将问题与答案论坛添加到页面 {#adding-a-q-a-forum-to-a-page}

要在创作模 `QnA` 式下将组件添加到页面，请使用组件浏览器在应 `Communities / QnA`当显示问题与答案论坛的页面上查找并将其拖动到相应位置。

有关必要的信息，请访 [问社区组件基础](/help/communities/basics.md)。

#### 设置选项卡 {#settings-tab}

在“设 **置** ”选项卡下，指定主题（问题）和回复（答案）的设置：

* **允许附加缩略图**

   如果选中此项，则会创建附加图像的缩略图。

* **附加缩略图时的大小上限**

   附件缩略图图像的最大大小（以像素为单位）。 默认值是800 x 800。

* **缩略图的最小图像大小**

   用于为内联图像生成缩览图的最小图像大小（以字节为单位）。 默认值是100000字节(100 kb)。

* **缩略图大小最大值**

   内联图像的缩略图最大大小（以像素为单位）。 默认值是800 x 800。

* **每页主题数**

   定义每页显示的问题／帖子数。 默认值为10。

* **已审核**

   如果选中此项，则发布主题和评论必须经过批准，才能显示在发布站点上。 取消选择默认值。

* **关闭**

   如果选中此项，则论坛将不会显示新的问题和评论。 取消选择默认值。

* **富文本编辑器**

   如果选中此项，则主题和注释可以输入标记。 取消选择默认值。

* **允许标记**

   如果选中此项，则允许成员向其帖子中添加标记标签(请参阅 **标记字段** 选项卡)。 取消选择默认值。

* **允许文件上传**

   如果选中此项，则允许向问题或评论添加文件附件。 取消选择默认值。

* **允许关注**

   如果选中此项，则为论坛帖子添加以下功能，允许成员获 [得](/help/communities/notifications.md) 新帖子的通知。 取消选择默认值。

* **允许固定**

   如果选中此项，则论坛主题可以固定到主题列表的顶部。 取消选择默认值。

* **允许电子邮件订阅**

   如果选中此项，则允许会员通过电子邮件([订阅](/help/communities/subscriptions.md))通知新帖子。 需要选中“允许跟随”并配置 [电子邮件](/help/communities/email.md)。 取消选择默认值。

* **最大文件大小**

   仅在选中时 `Allow File Uploads` 相关。 此字段限制已上载文件的大小（以字节为单位）。 默认为104857600(10 Mb)。

* **允许的文件类型**

   仅在选中时 `Allow File Uploads` 相关。 以逗号分隔的列表文件扩展名和“点”分隔符。 例如：.jpg、.jpeg、.png、.doc、.docx、.pdf。 如果指定了任何文件类型，则不允许上传那些未指定的文件类型。 默认值未指定，因此允许** **所有文件类型。

* **附加图像文件最大大小**

   仅在选中“允许文件上传”时相关。 已上载图像文件可以具有的最大字节数。 默认为2097152(2 Mb)。

* **允许回复**

   如果选中此项，则允许回复发布到问题的评论。 取消选择默认值。

* **允许投票**

   如果选中此项，则在问题中包含“投票”功能。 取消选择默认值。

* **允许用户删除评论和主题**

   如果选中此项，则允许成员删除其发布的评论和问题。 取消选择默认值。

* **允许拥有权限的成员**

   如果选中此项，则仅允许“特权”成员创建内容。

* **在作者编辑模式下阻止“用户生成内容”**

   如果启用，则在创作模式下编辑时阻止用户生成的内容。

* **将选定的答案移到顶部**

   如果选中此项，则显示的第一个答案为选定的答案。 取消选择默认值。
* **显示徽章**

   如果选中此项，则使用成员的 [博客条目](/help/communities/implementing-scoring.md) ，显示已获得和已分配的徽章。 取消选择默认值。

* **允许专题内容**

   如果选中此项，该构思将被识别为特色 [内容](/help/communities/featured.md)。 取消选择默认值。

* **启用提及功能**

   如果启用，允许注册的社区用户识别其他注册成员（使用名、姓、用户名），并使用通用的@user-name语法标记这些成员。 标记用户会收到有关其提及次数的通知。

* **最大提及次数**

   限制帖子中允许的最大提及次数。 默认值为10。

* **UI 提及模式**

   指定允许的模式字符串以标记帖子中的注册用户（@提及）。 For example, `~{{familyName}}{{givenName}}`.

#### “用户审核”选项卡 {#user-moderation-tab}

在“用户 **协调** ”选项卡下，指定如何管理已发布的主题（问题）和答案（用户生成的内容）。 有关详细信息，请参阅 [审核用户生成的内容](/help/communities/moderate-ugc.md)。

* **拒绝回答**

   如果选中此项，则允许受信任的成员审核者拒绝发布的答案，并阻止该答案显示在公共问答论坛上。 取消选择默认值。

* **关闭／重新打开主题**

   如果选中此项，则受信任的成员审核者可以关闭问题（主题）以进一步编辑和回答，还可以重新打开问题。 取消选择默认值。

* **移动主题**&#x200B;如果选中此项，则允许发布端版主移动问题。 取消选择默认值。

* **旗标帖子**

   如果选中此项，允许成员将其他人的问题或答案标记为不合适。 取消选择默认值。

* **旗标原因列表**

   如果选中此项，允许成员从下拉式列表中选择将问题或答案标记为不适当的理由。 取消选择默认值。

* **自定义标志原因**

   如果选中此项，则允许成员输入自己将问题或答案标记为不适当的原因。 取消选择默认值。

* **协调阈值**

   输入在通知版主之前必须由成员标记问题或答案的次数。 默认值为1（一次）。

* **标记限制**

   输入在问题或答案隐藏于公共视图之前必须标出的次数。 如果设置为-1，则标出的问题或答案永远不会隐藏在公共视图中。 否则，此数字必须大于或等于审核阈值。 默认为5。

#### 标记字段选项卡 {#tag-field-tab}

在“标 **记字段** ”选项卡下，可应用的标记（如果允许）会根据选择的命名空间 **** 来限制。

* **允许的命名空间**

   如果选中 `Allow Tagging` 了“设置”选项卡下 **的选项** ，则相关。 可以应用的标记仅限于选定的命名空间类别中的标记。 命名空间的列表包括“标准标记”(默认命名空间)和“包括所有标记”。 默认值为none checked，表示允许所有命名空间。

* **建议限制**

   输入要作为建议显示给发布到论坛的成员的标记数。 值**-**1表示没有限制。 默认值为0。

#### “排序设置”选项卡 {#sort-settings-tab}

在“排 **序设置** ”选项卡下，指定显示已发布注释时的排序方式。

* **排序方式**

   选中所有允许的排序选择： `Newest, Oldest, Last Updated, Most Viewed, Most Active, Most Followed and Most Liked`. Default is `Newest, Oldest, Last Updated`.

* **设置为默认**

   下拉以选择一个选中的排序选项作为默认值显示。 Default is `Newest`.

* **选择分析排序的时间选项**

   下拉以选择其中一个 `All, Last 24 Hours, Last 7 Days, Last 30 Days`。 Default is `All`.

## 站点访客体验 {#site-visitor-experience}

### 识别答案 {#identifying-answers}

一个答案可以使用按钮标记为正确或有用的 `Select Answer` 答案。 将某个问题标记为“已回答”后，只有在使用按钮取消选择第一个答案之前，才能选择另一个 `Unmark Chosen Answer` 答案。

选择可行答案后，可以使用按钮取消选择该 `Unmark Chosen Answer` 答案。

选择答案作为可行答案后，问题解答主页上的问题主题旁 `Answered` 会显示问题的指示。

#### 版主和管理员 {#moderators-and-administrators}

当登录用户具有审查方或管理员权限时，他们可以执行组件配置所允许的审核任务，而不管问题或答案的作者是谁。

他们还可以识别答案。

#### 成员 {#members}

站点访客登录后，根据配置，他们可以：

* 发布新问题。
* 编辑或删除他们创作的问题。
* 标记其他成员的问题或答案。
* 确定他们创作的问题的答案。

#### 匿名 {#anonymous}

未登录的站点访客只能阅读已发布的问题和答案，如果支持，可翻译这些问题和答案，但不能添加问题和答案，也不能标记其他人的帖子。
