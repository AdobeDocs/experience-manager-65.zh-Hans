# 工作分配功能 {#assignments-feature}

启用分配和 [启用目录](/help/communities/catalog.md) 组件浏览器中的社区组中的组件是 [启用社区](/help/communities/overview.md#enablement-community).

启用工作总揽组件允许当前已登录的社区成员（学习者）访问其分配的 [启用资源](/help/communities/resources.md).

从 [AEM Communities启用入门](/help/communities/getting-started-enablement.md) 教程，请参阅 [体验已发布的网站](/help/communities/enablement-published-site.md) 中。
