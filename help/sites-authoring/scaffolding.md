# 基架{#scaffolding}

有时，您可能需要创建大量共享相同结构但内容不同的页面。 通过标准AEM界面，您需要创建每个页面，将相应的组件拖动到页面上，并单独填充每个组件。

通过基架，您可以创建一个表单（基架），其中包含反映您想要用于页面的结构的字段，然后使用此表单可轻松基于此结构创建页面。

>[!NOTE]
>
>虽然有脚手架 [模式](/help/sites-authoring/author-environment-tools.md#page-modes) 之后，可以在经典UI中维护和创建基架以及从基架创建新页面。 参见 [基架](/help/sites-classic-ui-authoring/classic-feature-scaffolding.md) 有关详细信息，请参阅经典UI创作文档。
