# 触屏 UI 功能状态 {#touch-ui-feature-status}

从AEM 6.4开始 [已弃用经典UI](../release-notes/deprecated-removed-features.md). Adobe不会进一步增强经典UI，同时鼓励用户利用触屏UI中提供的强大新功能。

从版本6.0开始， AEM引入了一个新的用户界面，称为“触屏UI”（简称“触屏UI”），该用户界面与 [!DNL Adobe Experience Cloud] 以及整个Adobe用户界面指南。 在几乎实现了功能对等性的情况下，它已成为AEM中的标准UI，该界面采用了以桌面为导向的旧版界面，称为“经典UI”。

虽然触屏优化UI中存在大多数功能，但仍有一些功能尚未完整，且将在未来版本中添加。

以下列表显示了在AEM 6.5中实施的功能的当前状态。

有关升级到AEM 6.5的客户的建议，请参阅 [客户的用户界面推荐](/help/sites-deploying/ui-recommendations.md).

>[!NOTE]
>
>本页仅介绍与经典UI的功能对等性。 在触屏优化UI中添加的、唯一的、经典UI中不存在的功能未列出。

>[!NOTE]
>
>此列表力求完成，但并不详尽。

## 图例 {#legend}

* **完成**:该功能在触屏UI中完全可用。
* **大部分**:该功能在触屏UI中大多可用。
* **缺少**:触屏优化UI中不存在该功能，必须使用经典UI才能执行此操作。
* **已更换**:该功能已被替换为新实施（其工作方式不同）。
* **已删除**:触屏优化UI中不再存在该功能，也不会被替换。
